# The Thai 사용자 매뉴얼

## 1. 앱 소개

`The Thai`는 매장 운영을 위한 고객/예약/쿠폰 관리 앱입니다.

이 앱으로 할 수 있는 일:
- 고객 등록 및 검색
- 방문 이력 관리
- 쿠폰 충전/차감
- 예약 등록/수정/취소/노쇼 처리
- 전화 수신 시 고객 정보 확인
- 휴대폰 주소록과 앱 DB 동기화
- 쿠폰 차감 문자 발송

---

## 2. 처음 실행할 때

처음 실행하면 아래 권한을 요청할 수 있습니다.

- 알림 권한
  - 전화 수신/통화 종료 알림에 사용됩니다.
- 전화 권한
  - 전화가 왔을 때 고객 정보를 찾는 데 사용됩니다.
- 연락처 권한
  - 휴대폰 주소록 가져오기/동기화에 사용됩니다.
- SMS 권한
  - 쿠폰 차감 문자를 보낼 때 사용됩니다.

권한을 허용해야 전화 연동, 주소록 연동, 문자 발송 기능을 정상적으로 사용할 수 있습니다.

---

## 3. 기본 화면 구성

앱 하단에는 3개의 탭이 있습니다.

- `고객`
  - 전체 고객 목록
- `쿠폰 고객`
  - 쿠폰 잔액이 0원보다 큰 고객만 따로 표시
- `예약`
  - 날짜별 예약 목록

각 탭은 다시 들어갈 때마다 최신 정보로 새로고침됩니다.

---

## 4. 고객 등록

### 신규 고객 등록 방법
1. `고객` 탭으로 이동합니다.
2. 오른쪽 아래 `사람+` 버튼을 누릅니다.
3. 전화번호를 입력합니다.
4. 유입경로를 선택합니다.
5. 필요하면 메모를 입력합니다.
6. `저장`을 누릅니다.

### 고객명 규칙
고객명은 자동으로 아래 형식으로 생성됩니다.

`강서{등급}{유입경로}({주간횟수})({야간횟수}){전화번호끝4자리}`

예:
- `강서New마통(0)(0)1234`

### 등급 기준
- `New`: 1회 이하
- `N`: 2~4회
- `S`: 5~9회
- `G`: 10~19회
- `V`: 20~49회
- `VV`: 50회 이상

---

## 5. 고객 검색

고객 검색창에서는 아래 방식으로 검색할 수 있습니다.

- 이름 검색
- 전화번호 전체 검색
- 전화번호 일부 검색

예:
- `01012345678`
- `1234`
- `5678`
- `12345678`

하이픈(`-`) 없이 검색해도 됩니다.

---

## 6. 고객 상세 화면

고객 목록에서 고객을 누르면 상세 화면으로 들어갑니다.

상세 화면에서 할 수 있는 일:
- 기본 정보 확인
- 메모 수정
- 쿠폰 충전
- 방문 추가
- 방문 이력 수정
- 방문 이력 삭제
- 고객 삭제

### 쿠폰 충전
1. 고객 상세로 들어갑니다.
2. `쿠폰 충전` 버튼을 누릅니다.
3. 금액을 입력하고 저장합니다.

### 방문 추가
1. 고객 상세 상단의 방문 추가 버튼을 누릅니다.
2. 서비스를 선택합니다.
3. 쿠폰 차감 내용을 확인합니다.
4. 확인하면 방문 이력이 저장됩니다.

방문 추가 시 자동 처리:
- 방문 횟수 증가
- 주간/야간 횟수 증가
- 고객명 규칙 재계산
- 쿠폰 차감
- 필요 시 문자 발송

---

## 7. 예약 관리

### 예약 등록
1. `예약` 탭으로 이동합니다.
2. 오른쪽 위 `+` 버튼을 누릅니다.
3. 고객을 검색해서 선택하거나, 미등록 고객 전화번호를 직접 입력합니다.
4. 날짜/시간을 선택합니다.
5. 서비스, 예약 경로, 메모를 입력합니다.
6. `예약 저장`을 누릅니다.

### 예약 수정
1. 예약 목록에서 해당 예약의 `수정` 버튼을 누릅니다.
2. 날짜, 시간, 서비스, 경로, 메모를 수정합니다.
3. 저장합니다.

### 예약 취소
- `취소` 버튼을 누르면 예약을 취소 처리합니다.

### 예약 노쇼
- `노쇼` 버튼을 누르면 노쇼 처리합니다.
- 필요하면 쿠폰 차감 여부를 같이 선택할 수 있습니다.

### 예약 삭제
- `삭제` 버튼을 누르면 예약 내역을 삭제합니다.

---

## 8. 예약과 고객 정보 자동 연동

예약을 저장하면 고객 정보가 자동으로 반영됩니다.

자동 처리 항목:
- 방문 횟수 증가
- 주간/야간 횟수 증가
- 방문 이력 생성
- 고객명 규칙 갱신
- 서비스가 있으면 쿠폰 자동 차감

예약이 `취소`, `노쇼`, `삭제`되면:
- 증가된 방문 횟수 롤백
- 생성된 방문 이력 롤백
- 예약 시 차감된 쿠폰 복원

---

## 9. 추가결제필요 배지

서비스 금액이 고객 쿠폰 잔액보다 큰 경우:
- 쿠폰은 남은 금액만 전부 사용됩니다.
- 고객 쿠폰 잔액은 `0원`이 됩니다.
- 예약 카드에 `추가결제필요` 배지가 표시됩니다.

즉, 쿠폰만으로 서비스 금액이 모두 처리되지 않은 예약이라는 뜻입니다.

---

## 10. 쿠폰 차감 문자

쿠폰이 차감되면 고객에게 문자를 보낼 수 있습니다.

### 일반 차감 문자
쿠폰 차감 후 잔액이 남아 있는 경우 사용됩니다.

### 전액 소진 문자
서비스 금액이 쿠폰 잔액보다 커서 쿠폰을 모두 사용한 경우 사용됩니다.

전액 소진 시 고객은:
- 쿠폰이 0원이 되고
- 전액 소진 안내 문자를 받습니다

---

## 11. 문자 템플릿 수정

문자 템플릿은 `쿠폰 고객` 탭에서 수정할 수 있습니다.

### 수정 방법
1. `쿠폰 고객` 탭으로 이동합니다.
2. 상단의 `SMS` 아이콘을 누릅니다.
3. 수정할 템플릿을 선택합니다.
   - `일반 차감`
   - `전액 소진`
4. 내용을 수정하고 저장합니다.

### 사용 가능한 치환 변수

일반 차감 문자:
- `[M/d]` : 날짜
- `[service]` : 서비스 약어
- `[remaining]` : 남은 쿠폰 잔액

전액 소진 문자:
- `[M/d]` : 날짜
- `[service]` : 서비스 약어

기존 템플릿 호환용으로 `[약어]`, `[영어]`도 서비스 약어로 치환됩니다.

### 서비스 약어
- `타이 60분 -> T60`
- `타이 90분 -> T90`
- `아로마 60분 -> A60`
- `아로마 90분 -> A90`
- `크림 60분 -> C60`
- `크림 90분 -> C90`
- `스웨디시 60분 -> S60`
- `스웨디시 90분 -> S90`

---

## 12. 전화가 왔을 때

### 앱을 열어둔 상태
- 전화가 오면 고객 정보 오버레이가 뜹니다.
- 등록 고객이면 고객명, 방문 횟수, 쿠폰 정보를 볼 수 있습니다.
- 미등록 고객이면 신규 등록으로 바로 이동할 수 있습니다.

### 통화 종료 후
- `예약을 받으셨나요?` 확인 창이 뜹니다.
- `예`를 누르면 예약 등록 화면으로 이동합니다.

자동 입력되는 정보:
- 통화한 전화번호
- 통화 종료 시각
- 기존 고객이면 예약 경로

---

## 13. 주소록 가져오기

휴대폰 주소록의 고객을 앱 DB로 가져올 수 있습니다.

### 실행 방법
1. `고객` 탭으로 이동합니다.
2. 상단 `다운로드` 아이콘을 누릅니다.
3. 가져오기 방식을 선택합니다.
   - `전체 덮어쓰기`
   - `없는 고객만 추가`
4. 미리보기 결과를 확인합니다.
5. `가져오기`를 누릅니다.

### 가져오기 대상
주소록 이름이 `강서`로 시작하는 연락처를 우선 대상으로 사용합니다.

지원 예:
- `강서N마통(2)(1)5678`
- `강서N마통,스페셜(0)(1)1234`
- `강서-N-마통(1)(0)1234`
- `강서G로드5678`
- `강서...헬로...`
- `강서...네이버...`

### 진행 표시
가져오기 중에는 다음과 같은 진행 문구가 보입니다.
- `N/M개 분석 중...`
- `신규 고객 생성 중`
- `기존 고객 업데이트 중`
- `주소록 동기화 중`

---

## 14. 주소록 자동 동기화

앱 DB에서 고객 정보가 바뀌면 로컬 주소록도 같이 갱신됩니다.

동기화 시점:
- 고객 등록
- 방문 처리
- 예약 저장
- 예약 취소/노쇼/삭제

전화번호가 같으면 기존 연락처를 찾아 이름을 업데이트하고, 없으면 새 연락처를 만듭니다.

---

## 15. 쿠폰 고객 탭 사용법

`쿠폰 고객` 탭에서는:
- 쿠폰이 남아 있는 고객만 따로 확인할 수 있습니다.
- 문자 템플릿을 수정할 수 있습니다.

이 탭은 일반 고객 목록과 별개 화면일 뿐, 고객이 이동되는 구조는 아닙니다.

---

## 16. 운영 시 주의사항

- 전화, 연락처, SMS, 알림 권한을 허용해야 전체 기능이 정상 동작합니다.
- 주소록 연동은 실제 휴대폰 기기에서만 동작합니다.
- 백그라운드 수신 알림은 시스템 알림이라 위치를 앱이 자유롭게 바꿀 수 없습니다.
- 고객명은 수동으로 입력하는 개념보다 앱 규칙으로 자동 관리되는 값으로 이해하면 됩니다.

---

## 17. 추천 사용 순서

처음 사용하는 경우 아래 순서로 익히는 것이 가장 쉽습니다.

1. 권한 허용
2. 고객 탭에서 신규 고객 1명 등록
3. 고객 상세에서 쿠폰 충전
4. 방문 추가로 쿠폰 차감 확인
5. 예약 탭에서 예약 등록/수정/취소 확인
6. 쿠폰 고객 탭에서 문자 템플릿 확인
7. 고객 탭에서 주소록 가져오기 테스트

---

## 18. 자주 쓰는 기능 빠르게 찾기

- 신규 고객 등록: `고객 탭 -> 오른쪽 아래 +`
- 고객 검색: `고객 탭 상단 검색창`
- 쿠폰 충전: `고객 상세 -> 쿠폰 충전`
- 방문 기록: `고객 상세 -> 상단 방문 버튼`
- 예약 등록: `예약 탭 -> 오른쪽 위 +`
- 예약 수정/취소/노쇼/삭제: `예약 목록 카드 버튼`
- 문자 템플릿 수정: `쿠폰 고객 탭 -> 상단 SMS 아이콘`
- 주소록 가져오기: `고객 탭 -> 상단 다운로드 아이콘`
